# cool-land

cool-land

## 安装

```bash
pnpm i
```

## 运行

项目拉取下来后，需要先build一次，因为项目中有一些依赖是需要build后才能使用的。

```bash
pnpm run prebuild
```

### 客户端

```bash
pnpm run dev:client
```
启动mock数据.
```bash
pnpm run dev:db
```
### 服务端

```bash
pnpm run dev:server_nest
```

### 组件库

```bash
pnpm run dev:comp
```
